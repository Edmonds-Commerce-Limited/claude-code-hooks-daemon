# Release v2.17.3 - Upgrade Skill Fix

**Release Date:** 2026-02-27
**Type:** Patch Release

## Summary

v2.17.3 fixes the `/hooks-daemon upgrade` skill failing when invoked without an explicit version argument.

## Changes

### Fixed

- **Upgrade Skill Version String Bug**: Fixed the `/hooks-daemon upgrade` skill passing the literal string `"latest"` as the target version instead of resolving the actual latest release tag. When no version argument was provided, the wrapper script defaulted to `"latest"` which the daemon's upgrade script interpreted as a git tag name (which doesn't exist), rather than triggering its auto-detect logic. The invocation now conditionally omits the version argument, allowing the daemon script to auto-detect the latest tag.
